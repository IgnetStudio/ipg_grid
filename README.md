## Description

Simple page written in Pug.

---

## Installation

1. Install [node.js](https://nodejs.org/en/) & [yarn](https://yarnpkg.com/)

2. Clone repository

`git clone https://github.com/IgnetStudio/ipg_grid.git`

3. Install dependencies:

`yarn`

4. Run local server:

`yarn dev`

5. Watch changes in pug files:

`yarn gulp pug watch`
